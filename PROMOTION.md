# 🚀 Rich Brother: 데이터 기반 주식 분석 플랫폼

**Rich Brother**는 투자자가 데이터를 통해 시장의 흐름을 읽고, 확실한 근거를 바탕으로 의사결정을 내릴 수 있도록 돕는 강력한 주식 분석 솔루션입니다.

---

## ✨ 핵심 기능 (Key Features)

### 1. 📈 강력한 차트 분석 (Advanced Charting)
- **12개월 장기 추세 확인**: 최근 1년 치의 주가 흐름을 한눈에 파악할 수 있습니다.
- **이동평균선 완벽 통합**: 5일, 20일, 60일, 120일, 240일 이동평균선을 차트에 기본 탑재하여 지지/저항 및 추세 전환을 즉각적으로 포착합니다.
- **직관적인 거래대금 시각화**: 주가 등락에 따른 거래대금 색상 구분(상승시 적색, 하락시 청색)으로 매수/매도 강도를 쉽게 읽을 수 있습니다.

### 2. ⚡ 120일 돌파 전략 (120-Day Breakout Strategy)
- **추세의 시작점 포착**: 120일 이동평균선을 강하게 돌파하는 종목을 자동으로 발굴합니다.
- **V1 & V2 듀얼 모드**: 기본 돌파 전략(V1)과 더욱 정교한 필터링이 적용된 V2 모드를 제공하여 투자 성향에 맞는 종목을 선별할 수 있습니다.
- **상세 데이터 제공**: 시가총액, 등락률, 섹터 정보 등을 한 화면에서 확인 가능합니다.

### 3. 📊 종목별 수익률 상세 조회 (Deep Dive Inquiry)
- **다각도 수익률 분석**: 1일, 5일, 10일, 20일, 60일, 120일, 240일, 연초 대비(YTD), 작년 수익률까지 모든 기간의 성과를 분석합니다.
- **정밀 필터링 시스템**:
  - **종목/코드 검색**: 원하는 종목을 빠르게 찾습니다.
  - **시가총액 범위**: 대형주, 중소형주 등 원하는 규모의 종목만 골라냅니다.
  - **시장 구분**: 주식(코스피/코스닥)과 ETF를 구분하여 조회 가능합니다.
  - **수익률 조건**: "5일 수익률 10% 이상"과 같이 구체적인 조건으로 급등주를 찾아냅니다.

### 4. 🔄 스마트 데이터 동기화
- **자동화된 수집**: 매일 시장 데이터를 수집하여 최신 상태를 유지합니다.
- **대용량 처리**: 수천 개의 종목 데이터를 빠르고 안정적으로 처리합니다.

## 💻 기술 스펙 (Tech Stack)
- **Frontend**: React, TypeScript, TailwindCSS, Recharts, React Query
- **Backend**: FastAPI (Python)
- **Database**: Oracle Database (cx_Oracle)
